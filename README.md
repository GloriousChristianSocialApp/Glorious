# Glorious app

Packages ar managed in * **`./pubspec.yaml`**

# /lib
* **`./lib`** Contains source code
* **`./lib/models`** contains the defination of components
* **`./lib/widgets`** contains the actual widgets
* **`./lib/screens`** contains app screens
* **`./lib/services`** contains api handlers and other app required processing

# /web
Contains web bundler info
# /android
* **`./android`** Contains custom Android native code
# /ios
* **`./ios`** Contains Custom ios native code
# /macos
* **`./macos`** Contains custom macos code
# /windows
* **`./windows`** Contains custom Windows code 
# /linux
* **`./linux`** Contains custom Linux native code
